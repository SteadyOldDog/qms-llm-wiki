# BSL各级别实验室核心参数对比

## 概述

本文依据 [[GB 19489-2008 生物安全通用要求]]，对BSL-1至BSL-4各级别实验室的核心参数进行横向对比，便于快速查阅和决策。相关标准详见 [[GB 19489-2008 生物安全通用要求]]。

## 通风与负压对比

| 参数 | BSL-1 | BSL-2 | BSL-3 | BSL-4 |
|------|-------|-------|-------|-------|
| 通风方式 | 自然通风 | 无特殊要求 | **独立送排风系统** | 独立送排风系统 |
| 负压（对室外） | 无 | 无 | >=30~40Pa | **>=60Pa** |
| 负压（对相邻） | 无 | 无 | >=10~15Pa | **>=25Pa** |
| 换气次数 | 无 | 无 | **>=12次/h** | >=12次/h |
| 送风过滤 | 无 | 无 | HEPA | HEPA |
| 排风过滤 | 无 | 无 | HEPA | **两级HEPA** |
| 空气循环 | 不得循环 | 无特殊要求 | **不得循环** | 不得循环 |

## 设施设备对比

| 参数 | BSL-1 | BSL-2 | BSL-3 | BSL-4 |
|------|-------|-------|-------|-------|
| 门禁 | 可锁闭 | 进入控制 | 门禁系统 | **严格门禁+记录** |
| 门自动关闭 | 无 | 主入口+BSC间 | 所有门 | 所有门 |
| BSC | 操作气溶胶时 | **操作病原必须** | 必须 | Ⅲ级BSC |
| 高压灭菌器 | 必要时 | **必须** | 生物安全型 | 双扉生物安全型 |
| 备用电源 | 无 | 必要时 | BSC等>=**30min** | >=**30min** |
| 淋浴间 | 无 | 无 | 需要时 | **强制淋浴** |
| 化学淋浴 | 无 | 无 | 无 | 4.4.4类必须 |

## 围护结构对比

| 参数 | BSL-1 | BSL-2 | BSL-3 | BSL-4 |
|------|-------|-------|-------|-------|
| 地面 | 平整防滑 | 同BSL-1 | 防渗漏耐腐蚀 | 同BSL-3 |
| 窗户 | 防蚊纱窗 | 同BSL-1 | **密闭窗** | 密闭窗 |
| 缝隙密封 | 无 | 无 | **可靠密封** | 可靠密封 |
| 气密性 | 无 | 无 | 烟雾检测 | **500Pa->20min<250Pa** |

## 总结

BSL各级别是**递进关系**，核心差异：
1. **负压等级**：从无到>=60Pa
2. **HEPA过滤**：从无到两级过滤
3. **围护结构**：从普通到严格气密性
4. **门禁控制**：从锁闭到严格门禁+记录
5. **生命支持**：BSL-4独有
