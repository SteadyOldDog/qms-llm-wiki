# BSL-2实验室设施要求

## 概述

依据 [[GB 19489-2008 生物安全通用要求]] 第6.2章，BSL-2适用于能引起疾病但不严重、传播风险有限、有有效治疗预防措施的微生物。

## 设施要求（6.2.1~6.2.8）

| 条款    | 要求                                  |
| ----- | ----------------------------------- |
| 6.2.1 | 适用时，应符合 [[BSL-1实验室设施要求]] 的要求        |
| 6.2.2 | 主入口和BSC间门应可**自动关闭**；主入口应有**进入控制措施** |
| 6.2.3 | 工作区域外应有存放备用物品的条件                    |
| 6.2.4 | 应在工作区配备**洗眼装置**                     |
| 6.2.5 | 应配备**高压蒸汽灭菌器**或其他消毒灭菌设备             |
| 6.2.6 | 操作病原微生物的实验间应配备**生物安全柜**             |
| 6.2.7 | 按产品设计要求安装和使用BSC                     |
| 6.2.8 | 应有可靠的电力供应；重要设备必要时配置**备用电源**         |

## BSL-2 vs BSL-1 新增要求

| 要素   | BSL-1        | BSL-2           |
| ---- | ------------ | --------------- |
| 门控   | 可锁闭          | **自动关闭 + 进入控制** |
| 洗眼装置 | 操作刺激物时30m内设置 | **工作区必须配备**     |
| 灭菌设备 | 必要时配备        | **必须配备高压灭菌器**   |
| BSC  | 操作气溶胶时配备     | **操作病原微生物必须配备** |
| 备用电源 | 无要求          | **重要设备必要时配备**   |
