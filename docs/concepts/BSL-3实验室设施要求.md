# BSL-3实验室设施要求

## 概述

依据 [[GB 19489-2008 生物安全通用要求]] 第6.3章，BSL-3适用于能引起严重疾病、比较容易传播的微生物。是疾控中心实验室最常用的高等级实验室。

## 平面布局（6.3.1）

| 条款 | 要求 |
|------|------|
| 6.3.1.1 | 应明确区分**辅助工作区**和**防护区**，自成隔离区或独立建筑 |
| 6.3.1.2 | 核心工作间为直接从事高风险操作的工作间，人员通过缓冲间进入 |
| 6.3.1.4 | 4.4.2类：防护区至少含防护服更换间、缓冲间、核心工作间 |
| 6.3.1.6 | 传递窗应具备消毒灭菌条件 |

## 围护结构（6.3.2）

- 所有缝隙和贯穿处接缝应**可靠密封**
- 内表面应**光滑、耐腐蚀、防水**
- 地面应**防渗漏、完整、光洁、防滑、耐腐蚀、不起尘**
- 所有门应**自动关闭**
- 所有窗户应为**密闭窗**，玻璃应耐撞击、防破碎
- 通风空调正常运行时，烟雾测试应无可见泄漏

## 通风空调系统（6.3.3）

| 条款 | 要求 |
|------|------|
| 6.3.3.1 | 应安装**独立的送排风系统**，气流由低风险区向高风险区流动 |
| 6.3.3.3 | **不得循环使用**防护区排出的空气 |
| 6.3.3.5 | 送风应经过HEPA过滤器过滤 |
| 6.3.3.6 | 外部排风口与送风口直线距离>12m，高出建筑顶部>=2m |
| 6.3.3.8 | 应可在**原位**对排风HEPA过滤器进行消毒灭菌和检漏 |
| 6.3.3.10 | 送排风管道关键节点应安装**生物型密闭阀** |
| 6.3.3.12 | 应有**备用排风机** |

## 参数要求（6.3.10）

| 参数 | 要求 |
|------|------|
| 负压（对室外） | 4.4.1类：>=30Pa；4.4.2类：>=40Pa |
| 负压（对相邻区域） | 4.4.1类：>=10Pa；4.4.2类：>=15Pa |
| 最小换气次数 | >=12次/h |
| 温度 | 宜18~26℃（推荐性） |
| 相对湿度 | 宜30~70%（推荐性） |
| 噪声 | 安全柜开启时核心工作间<=68dB(A) |
| 洁净度 | 静态不低于8级 |

## 自控、监视与报警系统（6.3.8）

- 门禁系统：只有授权人员才能进入
- 启动顺序：**先排风，后送风**；关停顺序相反
- 排风故障时应避免实验室出现正压
- HEPA过滤器阻力连续监测
- 房间入口显著位置安装负压显示装置
- 中央控制系统实时监控、记录参数和故障
- 紧急报警应为**声光同时报警**

## 电力供应系统（6.3.6）

- BSC、送排风机、照明、自控系统、监视报警系统应配备**不间断备用电源**，至少维持**30min**
