# 实验室围护结构严密性检测与HEPA检漏方法

## 概述

[[GB 19489-2008 生物安全通用要求]] 附录A，为评价围护结构严密性和排风HEPA过滤器检漏提供参考。

## 方法一：烟雾检测法（A.2.1）

- 在通风空调系统正常运行条件下
- 通过人工烟源造成可视化流场
- 检测位置：接缝、门窗缝隙、插座、穿墙设备连接处
- 适用于BSL-3及以上日常检查

## 方法二：恒定压力下空气泄漏率检测法（A.2.2）

1. 温度控制在设计范围内并稳定
2. 关闭所有通路
3. 调节真空泵使房间产生并维持**250Pa负压差**
4. 测试不超过10min
5. 计算：Tf = Q / (V1 - V2)

## 方法三：压力衰减检测法（A.2.3）

1. 温度控制稳定，关闭所有通路
2. 使房间达到**500Pa负压差**
3. **每分钟记录1次压差和温度，连续记录至少20min**

> **判定标准（BSL-4）：** 500Pa后20min内衰减应小于250Pa

## HEPA扫描检漏方法（A.3）

- 检测用气溶胶中径通常为**0.3μm**
- 采样头距过滤器表面**2~3cm**
- 扫描速度<=**5cm/s**
- 扫描范围包括所有表面及连接处

## 应用场景

| 检测方法 | 适用级别 |
|----------|----------|
| 烟雾检测法 | BSL-3及以上 |
| 恒定压力泄漏率法 | BSL-3及以上 |
| 压力衰减检测法 | BSL-4 |
| HEPA扫描检漏 | BSL-3及以上 |
