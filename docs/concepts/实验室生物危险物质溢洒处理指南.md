# 实验室生物危险物质溢洒处理指南

## 概述

[[GB 19489-2008 生物安全通用要求]] 附录C，为实验室制定溢洒处理程序提供参考。

## 溢洒处理工具包（C.2）

| 物品 | 用途 |
|------|------|
| 消毒灭菌液 | 对感染性物质有效 |
| 镊子或钳子、可高压的扫帚和簸箕 | 处理锐器 |
| 足够的吸收材料 | 吸收溢洒物 |
| 专用收集袋或容器 | 盛放感染性溢洒物 |
| 橡胶手套、面部防护装备 | 个人防护 |
| 警示标识 | "禁止进入"、"生物危险" |

## 通用处理流程（C.3~C.4）

### 第一步：撤离房间
1. 通知无关人员离开，注意防护气溶胶
2. 关门并张贴警告标识
3. **至少30min后**方可进入处理
4. 如发生针刺或扎伤，挤压伤口周围促使血液外流
5. 如发生黏膜暴露，用水冲洗至少**15min**

### 第二步：清理溢洒区域
1. 穿PPE进入（需**两人**共同处理）
2. 用消毒灭菌剂浸湿的纸巾覆盖溢洒物
3. 从外围向中心倾倒消毒灭菌剂
4. 收集到专用容器中
5. 锐器用镊子或钳子处理

## BSC内溢洒处理（C.5）

- <1mL：直接用消毒灭菌剂浸湿的纸巾擦拭
- >=1mL：使BSC保持**开启状态**，覆盖吸收材料，消毒后清理
- **不要将头伸入安全柜内**

## 离心机内溢洒处理（C.6）

- 离心结束后至少等候**5min**再打开盖
- 破碎后立即关机不打开盖，切断电源，**至少30min后**开始清理
- 转子浸泡在非腐蚀性消毒灭菌液内建议>=**60min**
