# 加强型BSL-2实验室

## 定义

依据 [[WS 233-2017 病原微生物实验室生物安全通用准则]] 第6.3.2章，加强型生物安全二级实验室（Enhanced Biosafety Level 2 Laboratory）是在普通型BSL-2实验室的基础上，通过**机械通风系统**等措施加强实验室生物安全防护要求的实验室。

## 与普通型BSL-2的区别

| 要素 | 普通型BSL-2 | 加强型BSL-2 |
|------|-------------|-------------|
| 布局 | 无特殊要求 | **必须包含缓冲间和核心工作间** |
| 通风 | 无机械通风要求 | **必须采用机械通风系统** |
| 负压 | 无要求 | 核心工作间相对相邻区域为**负压**，压差宜>=10Pa |
| 排风过滤 | 无要求 | **排风系统应使用HEPA过滤器** |
| 门互锁 | 无要求 | 缓冲间门**宜能互锁**，设紧急手动解除开关 |
| 监控 | 无要求 | **应自动控制压力并可报警** |
| 温度噪音 | 无要求 | 核心工作间18~26℃，噪音<68dB |
| 灭菌器 | 必要时配备 | **实验室内应配置压力蒸汽灭菌器** |

## 设施要求（6.3.2.1~6.3.2.13）

| 条款 | 要求 |
|------|------|
| 6.3.2.1 | 适用时，应符合普通型BSL-2的要求 |
| 6.3.2.2 | 应包含**缓冲间和核心工作间** |
| 6.3.2.3 | 缓冲间可兼作防护服更换间；必要时设准备间和洗消间 |
| 6.3.2.4 | 缓冲间门宜能互锁，设**紧急手动互锁解除开关** |
| 6.3.2.5 | 设洗手池，水龙头开关应为**非手动式** |
| 6.3.2.6 | 采用机械通风，送排风口防雨防风防虫；排风使用**HEPA过滤器** |
| 6.3.2.7 | 核心工作间送排风口布置符合**定向气流**原则 |
| 6.3.2.8 | 核心工作间相对相邻区域为**负压**，压差宜>=10Pa；入口设**压力显示装置** |
| 6.3.2.9 | 自动控制压力稳定性，可**异常报警** |
| 6.3.2.10 | 排风与送风**连锁**，排风先于送风开启，后于送风关闭 |
| 6.3.2.11 | 围护结构应能承受送排风机异常时的压力载荷 |
| 6.3.2.12 | 核心工作间温度18~26℃，噪音<68dB |
| 6.3.2.13 | 配置压力蒸汽灭菌器及其他消毒设备 |

## 适用场景

加强型BSL-2适用于需要更高防护但仍可在BSL-2级别开展的实验活动，如：
- 操作可能产生气溶胶的病原微生物
- 大量病原微生物的培养和分离
- 需要负压防护但未达到BSL-3要求的实验活动

## 与其他要素的关系

- 是 [[WS 233-2017 病原微生物实验室生物安全通用准则]] 独有的概念
- [[GB 19489-2008 生物安全通用要求]] 中没有加强型BSL-2的分类
- 介于 [[BSL-2实验室设施要求]] 和 [[BSL-3实验室设施要求]] 之间
