# 生物安全防护水平分级

## 分级依据

依据 [[GB 19489-2008 生物安全通用要求]] 第4章：

| 等级 | 标识 | 适用微生物特性 |
|------|------|----------------|
| 一级 | BSL-1 | 通常不会引起人类或动物疾病的微生物 |
| 二级 | BSL-2 | 能引起疾病但不严重，传播风险有限，有有效治疗预防措施 |
| 三级 | BSL-3 | 能引起严重疾病，比较容易传播 |
| 四级 | BSL-4 | 能引起非常严重疾病，我国尚未发现或已消灭的微生物 |

## 实验室标识体系

| 标识 | 含义 |
|------|------|
| BSL-1~BSL-4 | 仅从事体外操作的实验室 |
| ABSL-1~ABSL-4 | 包括从事动物活体操作的实验室 |

## 实验室分类（4.4）

| 分类 | 描述 |
|------|------|
| 4.4.1 | 操作非经空气传播致病性生物因子的实验室 |
| 4.4.2 | 可有效利用BSC操作常规量经空气传播致病性生物因子的实验室 |
| 4.4.3 | 不能有效利用BSC操作常规量经空气传播致病性生物因子的实验室 |
| 4.4.4 | 利用具有生命支持系统的正压服操作经空气传播致病性生物因子的实验室 |

## 各级别核心参数对比

| 参数 | BSL-1 | BSL-2 | BSL-3 | BSL-4 |
|------|-------|-------|-------|-------|
| 负压（对室外） | 无 | 无 | >=30Pa | >=60Pa |
| 负压（对相邻） | 无 | 无 | >=10Pa | >=25Pa |
| HEPA过滤 | 不要求 | 不要求 | 送排风均需 | 送排风均需（排风两级） |
| 门禁 | 可锁闭 | 入口控制 | 门禁系统 | 严格门禁+记录 |
