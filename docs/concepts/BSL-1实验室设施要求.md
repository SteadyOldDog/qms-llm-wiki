# BSL-1实验室设施要求

## 概述

依据 [[GB 19489-2008 生物安全通用要求]] 第6.1章，BSL-1是最低防护级别，适用于操作不会引起人类或动物疾病的微生物。其要求构成所有更高级别实验室的基础。

## 门窗要求

| 条款 | 要求 |
|------|------|
| 6.1.1 | 门应有可视窗并可锁闭，门锁及门的开启方向应不妨碍人员逃生 |
| 6.1.10 | 如有可开启的窗户，应安装防蚊虫纱窗 |

## 洗手与更衣

| 条款 | 要求 |
|------|------|
| 6.1.2 | 应设洗手池，宜设置在靠近实验室出口处 |
| 6.1.3 | 门口处应设存衣或挂衣装置 |

## 建筑与装修

| 条款 | 要求 |
|------|------|
| 6.1.4 | 墙壁、天花板和地面应易清洁、不渗水、耐化学品腐蚀。**不应铺设地毯** |
| 6.1.5 | 台柜和座椅等应稳固，边角应圆滑 |
| 6.1.6 | 实验台面应防水、耐腐蚀、耐热和坚固 |
| 6.1.7~6.1.8 | 足够空间，合理摆放，避免交叉污染 |

## 通风与环境

| 条款 | 要求 |
|------|------|
| 6.1.9 | 可利用自然通风；机械通风应避免交叉污染 |
| 6.1.12 | 操作刺激或腐蚀性物质时，30m内设洗眼装置 |
| 6.1.13 | 操作有毒挥发物质时，配备负压排风柜 |

## 电气与给排水

| 条款 | 要求 |
|------|------|
| 6.1.16 | 应设应急照明装置 |
| 6.1.17~6.1.18 | 足够电力供应，可靠接地系统 |
| 6.1.19 | 供水和排水管道系统应不渗漏，下水应有防回流设计 |

## 应急与消毒

| 条款 | 要求 |
|------|------|
| 6.1.20 | 应配备适用的应急器材 |
| 6.1.21 | 应配备适用的通讯设备 |
| 6.1.22 | 必要时配备适当的消毒灭菌设备 |
