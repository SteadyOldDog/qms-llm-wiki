# GB 19489-2008 实验室生物安全通用要求

## 概述

中华人民共和国国家标准，规定了对不同生物安全防护级别实验室的设施、设备和安全管理的基本要求。

- **标准编号：** GB 19489-2008
- **替代标准：** GB 19489-2004
- **发布日期：** 2008-12-26
- **实施日期：** 2009-07-01
- **归口单位：** 全国认证认可标准化技术委员会（SAC/TC261）

## 核心特性

1. **区分技术要素与管理要素** — 第5-6章技术要素，第7章管理要素
2. **基于风险评估** — 整个标准以风险评估为基础和驱动
3. **覆盖BSL-1到BSL-4** — 包含体外操作和动物操作两类实验室
4. **大部分条款为强制性** — 仅6条为推荐性条款
