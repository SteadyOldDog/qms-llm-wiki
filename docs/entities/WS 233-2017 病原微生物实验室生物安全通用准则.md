# WS 233-2017 病原微生物实验室生物安全通用准则

## 概述

WS 233-2017《病原微生物实验室生物安全通用准则》是中华人民共和国卫生行业标准，规定了病原微生物实验室生物安全防护的基本原则、分级和基本要求。

- **标准编号：** WS 233-2017
- **替代标准：** WS 233-2002
- **发布日期：** 2017-07-24
- **实施日期：** 2018-02-01
- **发布机构：** 中华人民共和国国家卫生和计划生育委员会
- **强制性：** **全部为强制性条款**

## 核心特性

1. **全部强制性条款** — 与 [[GB 19489-2008 生物安全通用要求]] 不同，本标准无推荐性条款
2. **专门针对病原微生物** — 适用于开展微生物相关的研究、教学、检测、诊断等活动实验室
3. **独有加强型BSL-2** — 在普通型BSL-2基础上通过机械通风系统加强防护
4. **备案与认可制度** — BSL-1/2向市级卫生部门备案，BSL-3/4需通过国家认可
5. **菌（毒）种管理** — 有专门的管理制度，要求双人双锁、专人负责
